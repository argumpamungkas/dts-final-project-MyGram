# FinalProject
## Description
Final project of the Digitalent x Hacktiv8 Scalable Web Service with Golang bootcamp. MyGram is a RESTful API that enables users to create posts with images and captions, manage comments, and browse user feeds. Built with Gin Gonic, JWT authentication, PostgreSQL, and documented using Swagger.
